README — images/ (PlantUML sources)

Ce dossier contient les sources PlantUML pour les diagrammes destinés au manuscrit "PROGRAMMER SANS CODER".

Rendu recommandé :
- Installer PlantUML (https://plantuml.com/) et Graphviz
- Pour générer PNG : plantuml -tpng filename.puml
- Pour générer SVG : plantuml -tsvg filename.puml

Fichiers disponibles :
- architecture_overview.puml
- microkernel.puml
- data_pipeline.puml
- server_driven_ui.puml
- config_over_coding.puml
- plan_construction.puml
- deployment_k8s.puml
- ci_cd.puml
- prompt_architecture.puml

Après rendu, placez les PNG/SVG dans le même dossier et remplacez les placeholders [FIGURE: images/xxx.png] dans le document Word par les images correspondantes.
